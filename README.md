# yfinance_fixed

## Description
This package was developed to streamline fixing SSL errors encountered while using yfinance.

## Usage
import the package using the command:
```python
import yfinance_fixed as yfx
```

To use it, run:
```python
yfx.download(ticker)
```

You can also use the command with start and end date parameters if you wish:
```python
yfx.download(ticker, start='YYYY-MM-DD', end='YYYY-MM-DD')
```
